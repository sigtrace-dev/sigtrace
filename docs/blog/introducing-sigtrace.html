Introducing SigTrace: The Missing Debugger for Reactive Signals

By the SigTrace Contributors

Fine-grained reactivity is officially the new paradigm for state management in web development. With Angular introducing native Signals, Vue 3 solidifying Refs, and SolidJS building its entire architecture on createSignal, developer velocity has skyrocketed. But as systems grow, diagnosing the state flow becomes a significant challenge.

A single signal write can trigger a cascade of computed properties and run dozens of effects. When something breaks, finding the root cause becomes a tedious cycle of console.log() and breakpoints. Standard browser devtools help you inspect the final state, but they do not show the path that led there.

That is why we built SigTrace.

What is SigTrace?

SigTrace is an open-source, zero-configuration reactive lifecycle visualizer that runs directly inside Visual Studio Code. Rather than looking at a static snapshot of your state, SigTrace gives you a live telemetry stream of your reactive graph as you interact with your application.

Zero Refactoring Principle: SigTrace does not ask you to change your code. An AST compiler plugin automatically instruments your signals at compile time. Your production build remains completely untouched.

Key Features

1. Causal Chain Timelines

Whenever a signal write occurs, SigTrace captures the entire cascade of downstream reactivity in a chronological list. You can see the initial state trigger, followed by every computed memo re-evaluation and effect execution that ran in response — with microsecond-level timing overlays.

2. Click-to-Navigate Editor Sync

Since the debugger is embedded directly inside VS Code, every traced event is linked to its source code. Clicking any signal or computation in your activity table opens the file and highlights the exact line of code where it was declared.

3. Native RxJS Interop Support

In Angular, we frequently bridge RxJS streams and signals using toSignal(). SigTrace intercepts these streams at compile-time to log emissions, track value mutations, and link store selectors straight to your component view components.

Quick Start

Adding SigTrace to a project takes less than 2 minutes:

# Install dev dependencies
npm install --save-dev @sigtrace/core @sigtrace/vite

# Start your dev server with SigTrace instrumentation
npx sigtrace run ng serve  # Angular
# or
npx sigtrace run npm run dev  # Vue / SolidJS
